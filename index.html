BRAINSStandAlone

A thin wrapper project for holding onto several git submodules to ease building

Download this project as a .zip file Download this project as a tar.gz file

The BRAINSStandAlone is a harness to assist in building the many BRAINSTools under development.

Developers should run the "GitSetupEnvironmnet.sh" script to get started.

Example session for a clean build:

STEP 1: Download the source 

mkdir ~/source
cd ~/source
git clone git://github.com/BRAINSia/BRAINSStandAlone.git
cd BRAINSStandAlone/
STEP 2: Set up the build environment 
bash GitSetupEnvironmnet.sh
mkdir -p ../BRAINSStandAlone-build
cd ../BRAINSStandAlone-build/
STEP 3: Configure using CMake 
CC=/usr/bin/gcc-4.2 \
> CXX=/usr/bin/g++-4.2 \
> ccmake ../BRAINSStandAlone \
> -DUSE_BRAINSFit:BOOL=ON \
> -DUSE_BRAINSConstellationDetector:BOOL=ON \
> -DUSE_BRAINSABC:BOOL=ON
STEP 4: Build the project Use eight-core machine for building 
#!/bin/bash
NUMBEROFTHREADS=8
make -j${NUMBEROFTHREADS} -k
STEP 4a: Optional NOTE: The fetching of data may have problems with parallel builds, so you may need to restart it a few times 
make
